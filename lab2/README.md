# Лабараторная работа 2:

## Содержит структуру Track с 4 полями:

- Name (string) - название трека
- Duration (int) - продолжительность в секундах
- Genre (string) - жанр
- Rating (float64) - рейтинг


## Содержит тип перечисления RepeatMode с 3 значениями:

- RepeatNone (0) - повтор треков выключен
- RepeatOne (1) - повторяется текущий трек
- RepeatAll (2) - повторяются все треки по кругу

## Содержит структуру Playlist с 5 полями:

- Name (string) - название плейлиста
- tracks ([]*Track) - слайс указателей на треки
- currentIndex (int) - индекс текущего трека в слайсе
- repeatMode (RepeatMode) - режим повтора
- rng (*rand.Rand) - генератор случайных чисел для перемешивания

## Содержит 12 функций (методов):

- **NewPlaylist** - конструктор, создает новый плейлист
- **AddTrack** - добавляет трек в плейлист
- **DeleteTrack** - удаляет трек из плейлиста по названию
- **Shuffle** - перемешивает треки в плейлисте
- **SetRepeatMode** - устанавливает режим повтора
- **CurrentTrack** - получение текущего трека, возвращает указатель на экземпляр Track
- **NextTrack** - переключается на следующий трек (он становится текущим)
- **PrevTrack** - переключается на предыдущий трек (он становится текущим)
- **Display** - вывод плейлиста в виде таблицы с визуальным индикатором текущего трека (▶)
- **FindTracksInTimeRange** - поиск в плейлисте n треков с длительностью между указанными значениями, возвращает слайс с именами треков
- **SaveToFile** - сохраняет плейлист в текстовый файл (Base64 кодирование, одна строка — один трек)
- **LoadFromFile** - загружает плейлист из текстового файла (декодирование из Base64)
