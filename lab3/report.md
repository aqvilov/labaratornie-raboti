# Реализация стека и решение задачи на скобочную последовательность

## Реализованные структуры и функции

### 1. Стек (`Stack`)
- **Хранение данных**: `[]rune` (массив Unicode символов)
- **Основные методы**:
  - `Push(item rune)` - добавление элемента
  - `Pop() (rune, error)` - удаление и возврат верхнего элемента
  - `Peek() (rune, error)` - просмотр верхнего элемента без удаления
  - `IsEmpty() bool` - проверка на пустоту
  - `Size() int` - получение размера
  - `Clear()` - очистка стека

### 2. Проверка скобочных последовательностей

#### Основная задача (`isValid`)
- Поддерживаемые скобки: `()`, `[]`, `{}`
- Алгоритм:
  1. Открывающие скобки помещаются в стек
  2. При закрывающей скобке проверяется соответствие с верхушкой стека
  3. В конце стек должен быть пустым

## Тестирование
### Результаты проверки всех скобок
| Выражение | Результат | Ожидание | Статус |
|-----------|-----------|----------|--------|
| `()[]{}`  | true      | true     | ok     |
| `([)]`    | false     | false    | ok     |
| `{[]}`    | true      | true     | ok     |
| `(`       | false     | false    | ok     |
| `)`       | false     | false    | ok     |
| `` (пусто)| true      | true     | ok     |
